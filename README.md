# NoteTaker
NoteTaker is designed to help you manage your daily tasks, reminders, thoughts, and important ideas- all in one secure place.
Built using Servlets, JSP, and Hibernate, the app ensures speed, simplicity, and reliability. 
Created Add Note Page to add the newly notes and get it from Notes Directory.
Also Notes can be Searched from the Search Box by using title name.
In Notes Directory, created a set of 3 notes per page, with the help of Pagination in hibernate Configuration.
All over freely accessible webpage for all the users.
